# Prerequisites :
    - Java Application Server (Tomcat)
    - Java
    - IDE

# Installation tomcat : 
    - while using VSCODE :
        * install the comnuity server extension and install tomcat inside
        * install maven
    - while using eclipse : 
        just add your tomcat server inside the eclipse server

# Architecture Recap
    - Dynamic Web App (Eclipse):
        my-jsp-app/
        └── WebContent (or webapp)/
            └── index.jsp (Directly here)

    - Maven Web App (VS Code/Eclipse):
        my-jsp-app/
        └── src/
            └── main/
                └── webapp/
                    └── index.jsp (Must be in this specific sub-folder)

# URL & Context Path Logic
    - DWA: http://localhost:8080/my-jsp-app/
        (Mapped automatically by Eclipse)

    - Maven (VS Code): you must compile the project :
        `` mvn compile war:exploded ``
        - then run the   target/my-jsp-app ` to the server 
        - then : http://localhost:8080/my-jsp-app/

# 00
### JSP expressions (return sting like echo)
    ```
    <%= java code here %>
    ```    
    **example : <h2> <%= new java.util.Date() %> </h2>
### JSP Scriplets (local variables)
    ```
    <% 
        for (int i = 0; i < 5; i++) {
            out.println("Hello world");
        }
    %>
### JSP Declarations (global variables)
    ```
    <%!
        String var = "test";
        String makeItLower(String text) {
            return text.toLowerCase();
        }
    %>
    ```

# 01
### Call Java Class from JSP
    - create the folder /src/main/java
        - create a package : com.learn.jsp
            - create the java class : HelloWorld.java

    - create a hello_world.jsp in /src/main/webapp/WEB-INCF/views
        - just call the method like this ``package.class.method()`` : 
            ``` <%= com.leanr.jsp.HelloWorld.HelloWorldLowerCase() %>
            ```
        - you can also use :
            ``<%@ page import="com.learn.jsp.HelloWorld" %>`` and use direct without the package : 
                `` <%= HelloWorld.HelloWorldLowerCase() %> ``
